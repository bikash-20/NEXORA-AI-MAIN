# Nexora — Your AI Bestie ✨
Nexora is a futuristic, context-aware AI companion designed for students and developers. 

### 🚀 Features:
- **Bestie Personality:** Sassy, loyal, and full of "Banglish" charm.
- **Critical Math:** Solves engineering and physics problems.
- **Developer Tools:** String manipulation, code formatting, and word counting.
- **Local Context:** Knows the weather, traffic, and vibes of Dhaka.
